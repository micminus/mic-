Mic- - Doc

Le langage de programmation Mic- (extension : .micm) est créer dans l'optique suivante : apprendre et donner le gout a la programmation, ainsi qu'à être simple et lisible tout en restant extendable et personnalisable

Mot clé de base :

Pour les commentaires, utilisez le mot clé §.

Pour afficher du texte dans la console, utilisez le mot clé display.

Pour récuperer une valeur, utilisez le mot clé get suivi du type et du nom voulus.

Exemples :

  • § "Hello World"
  • display "Hello World"
  • get str 'x'

Déclarations de variables :

On utilise le mot clé hold pour les variables qui peuvent changer.

On utilise le mot clé store pour les variables qui ne peuvent pas changer.

Pour modifier des variables, on utilise le mot clé set.

Il existe plusieurs type de variables :

  • int pour les entiers
  • float pour les décimaux
  • str pour les chaines de caractères
  • char pour les caractères
  • bool pour les booleens (true/false)
  • list pour les listes
  • dict pour les dictionnaires

Exemples :

  • hold str y = "10" § y peut changer
  • store char z = 'a' § z ne peut pas changer
  • set y = "15" § y a changer

Loupes :

On utilise le mot clé iterate, with et as pour les loupes.

Exemples :

  • iterate range(1, 10) with x as str : display x
  • iterate while x < 10 : display x
Plus d'informations sur les loupes

Conditions :

On utilise le mot clé check pour les conditions.

Si la condition est fausse, on utilise le mot clé otherwise.

Et si la condition est fausse, mais que l'on a une autre condition, on utilise le mot clé otherwise check.

Exemples :

  • check x < 10 : display "x < 10" otherwise check x = 10 : display "x = 10" otherwise : display "x > 10"

Fonctions et Classes:

On utilise le mot clé structure pour les classes.

On utilise le mot clé process pour les fonctions.

Exemples :

  • process range(x, y) { display "x = " + x + ", y = " + y }
  • structure Point { process init(x, y) { self.x = x self.y = y } process toString() { display "x = " + self.x + ", y = " + self.y } } Point(1, 2).toString()

Libraries :

On utilise le mot clé load pour importer une librairie.

On utilise le mot clé share pour déclarer une fonction ou une classe dans une librairie.

Exemples :

  • load math local int x = math.random(1, 10)
  • process sayhello() { display "Hello World" } share sayhello
