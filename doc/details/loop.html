Loupe

Les loupes se forment de la manière suivante :

iterate range(1, 10) with x as str : display x

Le mot clé iterate est important, étant donné que c'est lui qui permet de boucler.

Le mot clé range permet de faire une boucle, mais peut être remplacer par un autre mot clé tel que :

  • while permet de faire une boucle tant que la condition est fausse

  • until permet de faire une boucle jusqu'a ce que la condition soit vrai

  • length(x) permet de faire une boucle sur la longueur d'un tableau

Le mot clé with permet de déclarer la variable unique a la boucle, as permet de donner le type de la variable.

Exemples :

  • iterate range(1, 10) with x as str : display x
  • iterate while x < 10 : display x
  • iterate until x < 10 : display x
  • iterate length(y) with x as str : display x
